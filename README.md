# Project_Santos

### High level overview:

For any given artist name: <br>
• gets their discography of studio albums, <br>
• gets lyrics for each song,<br>
• makes visualisations of the data (by time)<br>
   - number of songs
   - word clouds
   - word distribution
• analyzes sentiment of the lyrics
